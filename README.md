# Customer Behavior Data Analysis Project

## Overview

This project focuses on analyzing customer shopping behavior to uncover useful business insights. The workflow includes data loading, exploratory data analysis (EDA), data cleaning, SQL-based analysis using PostgreSQL, and building an interactive dashboard in Power BI.

The goal of this project is to demonstrate end-to-end data analysis skills including data processing, querying, visualization, and reporting.

---

## Dataset

The dataset contains information related to customer shopping behavior, including:

* Customer demographics (Gender, Age)
* Product categories
* Purchase amount
* Subscription status
* Shipping type
* Review ratings

This dataset helps analyze purchasing patterns, customer preferences, and revenue distribution across categories.

The following tools were used in this project:

---

## Project Workflow

### 1. Data Loading

The dataset was loaded into Python using Pandas for analysis.

### 2. Exploratory Data Analysis (EDA)

EDA was performed to understand:

* Data distribution
* Missing values
* Category patterns
* Customer purchase behavior

### 3. Data Cleaning

Data preprocessing included:

* Handling missing values
* Removing duplicates
* Correcting data types
* Preparing the dataset for analysis

### 4. SQL Analysis (PostgreSQL)

The cleaned dataset was imported into PostgreSQL and SQL queries were used to extract insights such as:

* Revenue by product category
* Customer segmentation
* Purchase trends
* Subscription behavior

### 5. Dashboard Development

An interactive Power BI dashboard was created to visualize key insights including:

* Number of customers
* Average purchase amount
* Average review rating
* Revenue by category
* Sales by category
* Subscription status distribution

## Key Results

Some important insights discovered during the analysis include:

* Clothing category generated the highest revenue.
* Most customers do not have an active subscription.
* Accessories and clothing dominate overall sales.
* Customer purchasing behavior varies across product categories.

These insights can help businesses optimize product strategy and marketing decisions.
